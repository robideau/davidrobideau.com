Rover

Github

---

This project is a bundle of software created for use with the iRobot Create. Rover contains basic function libraries for use with the on-board ATMega128 microcontroller, allowing for user control over movement, environment detection, audio output. Rover also allows for bluetooth control via a PuTTY terminal, and serial communication via the Atmel Studio development tool.

---

Rover was written in C with a few AVR Assembly edits here and there. This project was my first foray into embedded development. I submitted Rover in April 2015 alongisde Colin Farrell and Nicholas Boos for my CprE 288 class. Using this software, we were able to successfully guide our robot through a random, unseen obstacle course.
