MIPS CPU

Github

---

This is the architecture for a single-cycle MIPS CPU with in-order pipeline and data/hazard control written in VHDL using ModelSim. The CPU can handle a number of basic MIPS instructions and can parse an assembly program from binary code written to a text file.

---

I submitted this project in December 2015 for my processor architecture course along with Juan Venegas and Zach Brase.
